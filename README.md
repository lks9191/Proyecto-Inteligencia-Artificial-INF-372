# Proyecto-Inteligencia-Artificial-INF-372
Implementación de un sistema de diagnóstico médico basado en síntomas utilizando Bernoulli Naive Bayes, PCA y K-Means. Incluye la resolución de problemas de optimización clásica (N-Reinas con Simulated Annealing y N-Caballos) como parte del Proyecto Final de IA.
