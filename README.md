Proyecto Final de Inteligencia Artificial: Diagnóstico Médico y Optimización

Este repositorio contiene el desarrollo de un sistema de aprendizaje automático para la predicción de enfermedades y recomendación de especialistas basado en el Symptom to Doctor Dataset.

Características principales:

Diagnóstico: Clasificador Bernoulli Naive Bayes con validación robusta (100% Accuracy).

Análisis de Datos: Reducción de dimensionalidad con PCA y agrupamiento no supervisado con K-Means.

Optimización: Implementación de algoritmos heurísticos (Recocido Simulado) para el problema de las N-Reinas y enfoque combinatorio para los N-Caballos.
